# MLDM Masters Assignments (1–10)

This repository contains **10 practical MLDM (Machine Learning and Data Mining)** assignments implemented in Python.
Each assignment has:
- `main.py` (runnable program)
- `README.md` (objective + run steps)

## Assignment Index

1. **Data Preprocessing Pipeline**
2. **Simple & Multiple Linear Regression**
3. **Logistic Regression Classification**
4. **K-Nearest Neighbors (KNN)**
5. **Decision Tree vs Random Forest**
6. **K-Means and DBSCAN Clustering**
7. **Hierarchical Clustering**
8. **PCA for Dimensionality Reduction**
9. **Association Rule Mining (Apriori)**
10. **SVM with Hyperparameter Tuning**

## Quick Start

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Run any assignment:

```bash
python assignment_01/main.py
python assignment_10/main.py
```
